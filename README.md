# Taller Mecànic Aleke

Pàgina web profesional per al Taller Mecànic Aleke amb versió animada del vídeo promocional "Tu coche como nuevo en 30s".

## 📋 Contingut

- **index.html** - Pàgina principal del taller amb informació de serveis i contacte
- **video-animation.html** - Versió animada interactiva del vídeo promocional
- **invideo-ai-1080 Tu coche como nuevo en 30s 2026-04-13.mp4** - Vídeo original

## 🎨 Característiques

### Pàgina Principal (index.html)
- Disseny responsiu i modern
- Llistat de serveis
- Formulari de contacte
- Reproductor de vídeo integrat
- Enllaç a la versió animada

### Versió Animada (video-animation.html)
- **Animació Before/After** - Transformació del cotxe succa a net
- **Efectes professionals** - Brillo, transicions suaus
- **Estadístiques visuals** - Mostra els beneficis del servei
- **Característiques destacades** - Detall dels serveis
- **Botons interactius** - Call-to-action
- **Totalment responsiu** - Funciona a mòbils i ordinadors
- **Animacions CSS pures** - Sense dependències externes

## 🚀 Com usar

### Obrir localment
1. Descarrega els arxius
2. Obri `index.html` en un navegador web
3. O clica en "Veure versió animada interactiva" per veure l'animació

### Penjar a GitHub Pages
1. Crea un repositori a GitHub
2. Fes push d'aquests arxius
3. Activa GitHub Pages en les settings
4. La pàgina estarà disponible en `https://teu-usuari.github.io/nom-repo`
